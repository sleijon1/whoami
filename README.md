# Simon Leijon's Portfolio

A clean, minimal portfolio website built with Next.js and deployed to GitHub Pages.

## About

This is a personal portfolio showcasing my background, interests, and approach to software engineering. The site features a terminal-inspired design with a focus on clarity and directness.

## Live Site

Visit the portfolio: **[simonleijon.dev](https://simonleijon.dev)**

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: CSS
- **Deployment**: GitHub Pages
- **Hosting**: GitHub

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview static build
npx serve out
```

The site is exported as static HTML and deployed automatically via GitHub Actions on each push to main.
